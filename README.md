## ðŸš¦ Getting Started

1. **Clone the repo:**
   ```bash
   git clone https://github.com/Sujaltalreja04/Weblancer_AI.git
   cd Weblancer_AI
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run the development server:**
   ```bash
   npm run dev
   ```
4. **Build for production:**
   ```bash
   npm run build
   ```
5. **Preview production build:**
   ```bash
   npm run preview
   ```
